## Key Features

1. **Multi-Device Support**: Each user (staff or public) can register multiple devices
2. **Platform Tracking**: Supports iOS, Android, and Web platforms
3. **Automatic Cleanup**: Invalid/expired tokens are automatically removed
4. **Non-Blocking**: Notifications don't block main operations
5. **Secure**: JWT authentication for staff, token auth for public users
6. **Scalable**: Uses Firebase multicast for efficient delivery
7. **Reliable**: Error handling and graceful degradation
